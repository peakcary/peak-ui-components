# peak-ui-component

A Design UI library for React🎈

一个React UI组件库

## 📦 安装 Install
```shell
npm i peak-ui-component
```


## 🚩 示例 Usage

```js
import React from 'react';
import { Button } from "peak-ui-component";

class App extends React.Component {
  render() {
    return (
      <div className="App">
        <Button text={'-后缀信息'}></Button>
      </div>
    );
  }
}

export default App;
```
